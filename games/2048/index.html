How to Play

Goal: Slide tiles to combine matching numbers and reach 2048!

Controls
Swipe or use arrow keys to slide all tiles in one direction.
Merging
When two tiles with the same number collide, they merge into one with double the value.
Scoring
Each merge adds the new tile's value to your score.
Game Over
The game ends when no more moves are possible.
